# 🚀 OgarniamFinanse - Landing Page

Profesjonalny landing page dla aplikacji mobilnej OgarniamFinanse - zarządzanie finansami osobistymi i grupowymi.

## 🛠️ Tech Stack

- **Next.js 15** - React framework z App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Lucide Icons** - Beautiful icons
- **Vercel Analytics** - Performance tracking

## ✨ Features

- 🎨 Modern, responsive design
- ⚡ Optimized performance
- 📱 Mobile-first approach
- 🎯 SEO optimized
- 🚀 Fast deployment on Vercel

## 🏃 Quick Start

```bash
# Install dependencies
npm install --legacy-peer-deps

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Deployed on [Vercel](https://vercel.com) with automatic CI/CD from GitHub.
